# Robotic-Failure-Detection-Datasets-Benchmark
Awesome list of Robotic Failure Detection Datasets. They can serve as training data for failure detection models, or reward models, as well as benchmarks to evaluate them in a systematic and reproducible way.

## Benchmarks

| Dataset | Environment | File Type | Sample Size | Release Date | Note | Link |
|---------|-------------|-----------|-------------|--------------|------|
| UR5-Fail | Real robot arm (UR5) | Images | 940 samples | :white_check_mark: 2026 | execution and planning failures | [HuggingFace](https://huggingface.co/collections/paulpacaud/ur5-fail-robot-failure-detection-benchmark) |
| BridgeDataV2-Fail | Real robot arm (WidowX250) | Images | 16K samples | :white_check_mark: 2026 | execution and planning failures | [HuggingFace](https://huggingface.co/collections/paulpacaud/bridgedatav2-fail-robot-failure-detection-benchmark) |
| RLBench-Fail | Simulator robot arm (Franka Emika Panda) | Images | 21K samples | :white_check_mark: 2026 | execution and planning failures | [HuggingFace](https://huggingface.co/collections/paulpacaud/rlbench-fail-robot-failure-detection-benchmark) |
| RoboFail | Real robot arm (UR5) | Images, Video | 183 samples | :white_check_mark: 2023 | execution and planning failures | [Personal website](https://www.cs.columbia.edu/~liuzeyi/reflect_data/) |
| AHA | Simulator robot arm (Franka Emika Panda) | Images | 21K samples | :x: | execution and planning failures | None |
