# Robotic-Failure-Detection-Datasets-Benchmark
Awesome list of Robotic Failure Detection Datasets. They can serve as training data for failure detection models, or reward models, as well as benchmarks to evaluate them in a systematic and reproducible way.

## Benchmarks

| Dataset | Environment | File Type | Sample Size | Release Date | Note |
|---------|-------------|-----------|-------------|--------------|------|
| UR5-Fail | Real robot arm (UR5) | Images | 940 samples | :white_check_mark: 2026 | execution and planning failures |
| BridgeDataV2-Fail | Real robot arm (WidowX250) | Images | 16K samples | :white_check_mark: 2026 | execution and planning failures |
| RLBench-Fail | Simulator robot arm (Franka Emika Panda) | Images | 21K samples | :white_check_mark: 2026 | execution and planning failures |
| RoboFail | Real robot arm (UR5) | Images, Video | 183 samples | :white_check_mark: 2023 | execution and planning failures |
| AHA | Simulator robot arm (Franka Emika Panda) | Images | 21K samples | :x: | execution and planning failures |
